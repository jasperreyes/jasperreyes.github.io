Circles


This exercise and exploration in CSS animations was inspired by an animated illusion found on YouTube, posted by brusspup.

Illusion

This series of dots seemingly moves in a curved direction, but all are actually moving in straight lines.

Once achieved, I found that slightly increasing or decreasing the compounded delay times led to new and interesting effects.

Switching Sides

compounded delay time: 0 milliseconds

Curl

compounded delay time: 62.5 milliseconds

Opening Circle

compounded delay time: 100 milliseconds

Rotating Circle

compounded delay time: 125 milliseconds

Sloppy Circle

compounded delay time: 187.5 milliseconds

Clover

compounded delay time: 250 milliseconds

Center Pop

compounded delay time: 500 milliseconds

Rotating Squares

compounded delay time: 875 milliseconds

Application

Width additional styling, these dynamic animations can potentially be used as loading animations for web/mobile interfaces.
