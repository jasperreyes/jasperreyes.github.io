About this project

Thanks for playing with Zwerp! This experiment started by playing with web animations, but expanded into an interactive experience using JS and jQuery, content writing, sound design, voice command APIs, and the subtlties of human facial expressions. You can read more about the design process here.

Please message me any thoughts and/or suggestions for improvement. Thanks!
